# NVS Transport — site vitrine (React + Vite)

## Démarrage
```bash
npm install
npm run dev
```

## Build (GitHub Pages)
```bash
npm run build
npm run preview
```

### Déploiement GitHub Pages (option simple)
1. Crée un repo GitHub, push ce projet.
2. Dans GitHub → **Settings → Pages** :
   - Source: **GitHub Actions** (recommandé) ou **Deploy from a branch**.
3. Si tu veux un workflow GitHub Actions, ajoute un job `vite build` + publication de `dist/`.
